# Khosla Uplift Pressure Calculator

This software performs Khosla-based seepage, uplift pressure, required floor thickness, and exit-gradient analysis for hydraulic structures founded on pervious foundations.

## Main features
- Calculation of pressure percentage values at E, D, and C points
- Mutual interference correction
- Floor-thickness correction
- Uplift pressure calculation
- Required floor thickness check
- Exit-gradient calculation
- 2D and conceptual 3D visualization
- Excel and PDF report generation

## Installation
pip install -r requirements.txt

## Running the software
python khosla_app/main.py
